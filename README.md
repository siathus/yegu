# 예약구매 프로젝트
## 프로젝트 소개
- 개발 기간 : 2024. 04 ~ 진행 중
- 특정 시간에 오픈하는 상품 예약 구매 서비스
- 대규모 트래픽 처리를 위한 MSA 채택

---

## 주요 기능
1. 회원 가입 및 로그인 기능
   - 회원 정보 암호화하여 적용
2. 상품 주문 기능
   - 상품 재고를 Redis로 관리하여 실시간 적용

---

## 개발 환경
- Java 21
- IntelliJ IDEA
